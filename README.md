##setup
`git clone`
`npm install`

#### auth
1. Log in to nytimes crossword site.
2. Go to a valid puzzle json url like http://www.nytimes.com/svc/crosswords/v2/game/12503.json
3. In network panel, copy your Cookie: header and update the script with it
